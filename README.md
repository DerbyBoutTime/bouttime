## bouttime

> 'Bout time we got this game started.

### Dependencies

* postgresql ~> 9.3
* ruby = 2.1.4

#### homebrew

```bash
./bin/homebrew.sh
```

#### laptop

If you don't have anything set up on your laptop you can take advantage of [laptop](https://github.com/wiserstudios/laptop). It supports Mac OS X and Linux.

### Setup

```bash
bundle install
```

### Deployment

* packer ~> 0.7
